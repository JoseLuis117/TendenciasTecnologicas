Tendencias Tecnológicas

Computación de usuario final (EUC)

1. Desktop as a Service (DaaS): Es un servicio en la nube que permite acceder a un escritorio virtualizado desde cualquier dispositivo con conexión a internet.

2. Remote Application Server (RAS): Es una solución que permite a los usuarios acceder a aplicaciones y datos alojados en servidores remotos, a través de una conexión de red.

3. Skeuomorphic design: Es un estilo de diseño que imita la apariencia y el comportamiento de objetos físicos en la interfaz de usuario de un dispositivo digital.

4. Storytelling: Es una técnica para comunicar información de manera efectiva a través de una narrativa estructurada.

5. Voice-User Interfaces (VUIs): Son interfaces de usuario que permiten la interacción con dispositivos o sistemas utilizando la voz como entrada.

6. Personal User Interfaces (PUIs): Son interfaces de usuario personalizadas que se adaptan a las preferencias y necesidades de cada usuario.

7. Invisible Design Systems: Son sistemas de diseño que se enfocan en crear experiencias de usuario intuitivas y sin distracciones, utilizando elementos y acciones minimalistas.

8. Micro interactions: Son pequeñas animaciones o efectos visuales que ocurren en la interfaz de usuario en respuesta a una acción del usuario.

9. Neumorphism: Es un estilo de diseño que utiliza efectos de iluminación y sombras para crear una apariencia tridimensional y realista en elementos de la interfaz de usuario.

10. Mobile Augmented Reality: Es una tecnología que combina la realidad aumentada con dispositivos móviles, permitiendo la superposición de elementos digitales en el mundo real a través de la cámara del dispositivo.
